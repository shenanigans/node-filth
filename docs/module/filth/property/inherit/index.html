inherit (

Perform a simple prototype inheritence operation to cause the first argument to inherit from the last. Unlike util.inherits the child prototype's own properties are copied over, so you can call inherit after setting up the prototype on child without anything getting left behind.

Arguments

| Function child

The class which will inherit from parent. Its prototype will be replaced with one chained to parent and own properties will be transfered over.
| Function parent

The class which will modify child. It is used to generate a new prototype for child which is chained to parent and its ancestors.
